# 🤖 Simple Agent Demo - ReAct 模式

一个精简的 AI Agent 项目，使用 LangGraph 和 Claude API 构建基于 ReAct 模式的 Agent。

## ✨ 核心特性

- **LangGraph Agent** - 使用 LangGraph 构建的 ReAct 模式 Agent
- **工具调用** - 5 个预定义工具（天气、搜索、翻译、时间，获取名字）
- **长期记忆** - 使用 JSON 文件存储对话和摘要
- **实时过程展示** - 前端实时展示 Agent 的思考、行动、观察过程
### 1. 安装依赖

```bash
cd simple-agent-demo
pip install -r requirements.txt
```
